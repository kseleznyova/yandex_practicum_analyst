# Интернет-магазин «Стримчик»
## Cтатус проекта: завершен
## Цель:
Необходимо выявить определяющие успешность игры закономерности.

Задачи:

- подготовить данные;
- провести исследовательский анализ данных;
- составить портрет пользователя каждого региона;
- проверить гипотезы.

## Инструменты:
- pandas 
- seaborn 
- scipy 
- plotly
- numpy
